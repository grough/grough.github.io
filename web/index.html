Hi, this is Gavin Rough. I'm interested in digital sound, computer art, and programming. Here are some links:

Creative coding with cellular automata

github.com/grough/processing-cellular-automata

This library simplifies the process of using cellular automata to make animated art.

Cubehelix for creative coding

github.com/grough/processing-cubehelix

Cubehelix is a colour scheme used by astronomers to render images of deep space. I didn't invent it, but I think it looks nice, so I made a version that can be easily used for creative coding.

Looping sampler for virtual modular synths

library.vcvrack.com/LilacLoop

Something like a looper pedal made to be used in a virtual modular synthesis environment. It works in stereo and can be configured to record up to 32 channels.

Pingverb virtual instrument

grough.com/pingverb

A simple virtual instrument that pings a reverb effect with differently flavoured impulses.

Live stream archive

youtube.com/@gavinrough

An archive of occasional improvised sound design sessions. For my own reference, but worth sharing.
